2) Page Loader Is Added As New Switcher Option.
---------------------------------------------------------------------------------------------
    Now you can also use page loader if you want, to enable page loader please follow below steps

    Step-1: Add loader="enable" attribute to the html tag (Path:Html/src/html/partials/mainhead.html)
    Step-2: Add new partial loader.html file in partials folder (Path:Html/src/html/partials/loader.html)
    Step-3: Include @SPK@include("partials/loader.html") in every html page after @SPK@include("partials/switcher.html") Partial.
    Step-4: Replace "assets" folder .
